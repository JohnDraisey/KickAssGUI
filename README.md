Kick Ass GUI is a GUI front end for SolidAngle's amazing Arnold Renderer. http://www.solidangle.com for more info.
FYI, this application is mostly useless unless you have Arnold.

2: Open src/KickAssGUI.pro

3: Ctrl/Command + r
